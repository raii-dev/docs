## Creating a Section

Sections help organize UI elements by grouping related components together inside a tab. You can create a section using the `createSection` function.

### Parameters

- **Name** (string): The title of the section that will be displayed.

### Example Usage

Here's how to create a section in `raii.lib`:

```lua
local mainSection = mainTab:createSection("This is the main section")  -- Creates a section
```
