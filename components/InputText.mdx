## Creating an Input Text Field

Input text fields allow users to enter and submit text. You can create one inside a tab using the `createInputText` function.

### Parameters

- **Name** (string): The label displayed next to the input field.
- **Callback** (function): A function that runs when the user submits input. The entered text is passed as a parameter.

### Example Usage

Here's how to create an input text field in `raii.lib`:

```lua
local inputText = mainTab:createInputText({
	Name = "This is an input text", -- Label displayed next to the input field
	Callback = function(inputText)
		if inputText ~= nil then
			print(inputText)
		end
	end,
})
