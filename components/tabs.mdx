## Creating a Tab

Tabs help organize different sections of your UI within the same window. You can create tabs using the `createTab` function.

### Parameters

- **Name** (string): The name of the tab that will be displayed.
- **Icon** (string): The lucide icon that will be displayed, if non it wont show any by default.
